Реализирайте клас Форма, който представлява списък от HTML елементи,
всеки от които съдържащ текстово поле.

Елементите са следните:

	- Button (4)
	- Text (1) - съдържа атрибута align,
	  който указва какво подравняване има текста: left, right или middle		
	- Password (2) - чийто атрибут е required,което показва дали е задължително попълването		
	- Check box (3) - има атрибутит checked,от който зависи дали отметката на полето е сложена 

Пример :

Следният текстов файл е пример за html документ.
								разяснения:
<!DOCTYPE html>
<html>
<body>

<form>
(1)	User : <input type="text" align="middle"> </br> 	- полето текст е преди <input>
(2)	Password : <input type="password" required> </br>	- полето текст е преди <input>
(3)	<input type="checkbox" checked> I agree ! </br>		- полето текст е след <input>
(4)	<input type="button" value="Submit"> </br>		- полето тексте в атрибут value
								- checked и required присъстват 
								  само ако съответната опция е активирана
</form>

</body>
</html>


Класът Форма е предвиден за генериране на html документ чрез C++ код. 

Функционалността, която трябва да реализирате е :
 
* добавяне на нов елемент от изброениете
* скриване на елемент по индекс (не се включва при генерирането на HTML-а)
* търсене на елемент по даден текст във формата.
  Ако такива елементи се съдържат да се върне индекса на първия.
* достъп до произволен елемент по индекс 
* запис на формата в текстов файл с разширение .html
 
Демонстрирайте използването на Формата в проста main функция.

Hints : -всеки елемент на формата може да има метод convertToHtml , който връща низ - 
        описанието на съответната форма в html

       -при правилна работа на Вашата програмата, 
        ще можете да визуализирате генерирания файл чрез браузера Ви
